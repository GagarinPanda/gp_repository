# Анализ рынка общественного питания г.Москвы

## Цель исследования

Подготовка анализа рынка общественного питания г.Москвы для предполагаемых инвесторов в небольшое кафе с роботами-официантами.

## Описание данных

Открытые данные о заведениях общественного питания г.Москвы (https://data.mos.ru/opendata/7710881420-obshchestvennoe-pitanie-v-moskve).

Таблица *rest_data*:

`id` — идентификатор объекта;

`object_name` — название объекта общественного питания;

`chain` — сетевой ресторан;

`object_type` — тип объекта общественного питания;

`address` — адрес;

`number` — количество посадочных мест.

## Ход исследования

1. Загрузить данные и подготовить их к анализу (тип данных, дубликаты, пропущенные значения).
2. Провести анализ данных:
- исследовать и построить графики соотношения видов объектов общественного питания, сетевых и несетевых заведений по количеству;
- определить вид объекта общественного питания для которого характерно сетевое распространение;
- дать харакетристику сетевым объектам по количеству посадочных мест на заведение;
- описать и построить график отображающий среденее количество посадочных мест для каждого объекта питания, определить вид объекта с самым высоким указанным показателем;
- найти и визуализировать топ-10 улиц по количеству размещённых на них объектов питания, определить районы Москвы в которых находятся данные улицы;
- найти число улиц с одним объектом общественного питания, определить районы Москвы в которых находятся данные улицы;
- выявить закономерности распределения посадочных мест для улиц с большим количеством объектов общественного питания;
- на основании проведенного анализа дать рекомендации о виде количестве посадочных мест, а также  районе расположения будущего заведения.
3. Подготовить презентацию для инвесторов по выполненному исследованию.
